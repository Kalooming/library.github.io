# Philoskus — Static Philosophy Site (starter)

This repository contains a minimal static site for publishing philosophy essays and notes.

Structure
- index.html — homepage
- about.html — about page
- posts/ — directory for essays (HTML files)
- assets/css/styles.css — styles and theme toggle
- assets/images/ — optional images and social preview

Deployment
- If the repository is named `Philoskus.github.io` and is in your personal account, GitHub Pages will publish from the default branch (usually `main`) at `https://philoskus.github.io/`.
- To publish: commit & push files to the repo root on the main branch.

Add a post
- Create an HTML file in `posts/` following the example `2025-12-25-introduction.html`.
- Link to it from the homepage or create an index of posts.

Optional enhancements
- Convert to Jekyll / Eleventy for templating and markdown support.
- Add comments via Utterances (GitHub issues) or Staticman.
- Add an RSS feed generator or small script to build one from posts, or enable Jekyll to do it.
